Introduction

Devin has ~7,000 tweets from 2012 - 2018 (6 years). This project analyzes all ~7,000 tweets, removing retweets and replies. The tweets are split into a bag of words where a term frequency analysis is performed to find Devin's top words. The same analysis is performed on 10 other Twitter users. We trained a model to predict the user that Devin tweets the most similar to in order to detect whether this can predict related goals, aspirations, interests, etc. We also perform a Digram Markov Chain on Devin's tweets in order to predict future tweets.

  • Markov

Problem Statement

We wanted to see if it were possible to detect similar people based on their tweets. We used Twitter's API (Tweepy) to pull the tweets. After extracting and cleaning the tweets, we performed K Nearest Neighbors to find Devin's most similar tweeters.
We hypothesized that Devin would tweet most similarly to Chrissy Teigen and Common White Girl. Take a look to see if we were correct.
